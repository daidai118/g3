# 世界观与设定 —— 灰烬之境 / Ash Realm

> 来源:从 `game1`(《灰烬之境》)提取精华,保留对**横版 2D** 项目有用的部分。
> 注:game1 后期转成了俯视角,本文已剔除俯视角专属内容,只留世界观 + 可直接复用于横版的设定。
> 配套文档:`动作游戏设计文档.md`(战斗 / 手感 / 系统)。

---

## 0. 一句话核心

> 在被灰烬吞噬的**艾什拉尔(Ashlar)**,失去记忆的**灰化者**追寻"灰烬之心",试图弄清灰烬到底是灾难、进化,还是又一次轮回的开始。

不是英雄、不是救世主,只是消亡世界里无数迷失灵魂之一。来这里不是为了拯救世界,而是为了找回自己的记忆。

---

## 1. 五条必须保留的核心体验

1. **灰烬不是单纯灾难** —— 它毁灭肉体与记忆,但也能被引导为力量;既是腐败,也是进化可能。
2. **记忆比生命更珍贵** —— 灰化者真正怕的不是死亡,而是遗忘。玩家通过环境、物品、NPC 碎片拼回真相。
3. **火种是绝望里的小亮点** —— 篝火、炉火、NPC 的坚持、琥珀色眼光,是灰暗世界里少数的暖色。
4. **技能既是钥匙也是武器** —— 每个能力都同时服务战斗与探索(钩索拉敌 / 圣光照明 / 震地破墙)。
5. **区域不是关卡,是世界的伤口** —— 每个区域代表艾什拉尔崩坏的一种形式。

---

## 2. 世界与历史

**艾什拉尔**:曾由神圣王权统治的繁荣文明,现处灰烬纪元末期,90% 地表被灰烬覆盖。
世界核心是一个沉睡的古老存在,它的"呼吸"形成灰烬;初火之王曾以生命封印核心。

| 时代 | 核心事件 | 留给游戏的内容 |
|---|---|---|
| 创世纪元 | 世界核心苏醒,第一次灰烬降临 | 化石、原始灰烬层、核心崇拜 |
| 火之纪元 | 初火之王发现火能净化灰烬 | 火焰、篝火、净化仪式 |
| 黄金纪元 | 艾什拉尔王国建立 | 王城、神殿、学院、旧贵族符号 |
| 衰落纪元 | 过度开采灰烬之心,封印松动 | 矿坑、禁忌研究、灰烬病 |
| 焚毁纪元 | 初火之王牺牲封印核心 | 牺牲、王之碎片、火种传说 |
| 灰烬纪元 | 封印瓦解,文明崩塌 | **当前游戏舞台** |

**核心谜团**:灰烬之心是什么?封印为何失效?玩家是普通灰化者,还是初火之王的灵魂碎片?灰烬该被净化、拥抱,还是打破轮回?

## 4. 区域(横版关卡蓝图)

```
初始墓地
  ├─ 腐朽森林
  ├─ 沉没教堂
  └─ 灰烬矿坑
        ↓
灰烬主城(安全区 / 情感锚点)
  ├─ 王城废墟
  ├─ 深渊回廊
  └─ 火焰神殿
        ↓
灰烬之心(终局 / 结局分歧)
```

| 区域 | 主题 | 画面关键词 | 叙事作用 |
|---|---|---|---|
| 初始墓地 | 苏醒、遗忘 | 墓碑、雾、枯树、破教堂远影 | 从死亡世界中醒来(教学区) |
| 腐朽森林 | 自然腐化与重生 | 黑色树根、幽绿孢子、枯萎巨树 | 灰烬对生命的侵蚀 |
| 沉没教堂 | 失落信仰 | 水淹礼拜堂、破彩窗、倒塌圣像 | 焚余教团与信仰崩塌 |
| 灰烬矿坑 | 挖掘与贪婪 | 竖井、矿车、毒气、发光灰脉 | 人类如何触碰世界核心 |
| 灰烬主城 | 残存文明 | 篝火广场、铁匠铺、塔楼、远景废墟 | 安全区与情感锚点 |
| 王城废墟 | 权力崩塌 | 王座、血池、碎旗、金色残辉 | 王室诅咒与初火历史 |
| 深渊回廊 | 虚空与真相 | 黑暗走廊、紫光裂隙、空间错位 | 接近被隐藏的答案 |
| 火焰神殿 | 信仰与净化代价 | 熔火、冰霜、祭坛、古火 | 火也不是免费救赎 |
| 灰烬之心 | 真相与选择 | 灰烬风暴、世界核心、记忆碎片 | 终局与结局分歧 |

> **横版利好**:墓地远处可见教堂剪影、主城远景废墟——多层视差背景天然契合"世界是一个整体"的叙事。这正好对接你 game3 brainstorm 里的"前中后景"需求。

## 5. 记忆即进度(替代等级系统)

成长不靠数值升级,而靠**发现世界真相**。每块记忆碎片是一把钥匙——解锁能力、剧情、新区域。

- 没有任务标记,没有经验值。驱动力是好奇心:"那座教堂里有什么?"
- 记忆碎片发现时用"水彩绽放"效果:琥珀色像水彩向外晕染,浸染周围灰色。
- 死亡后在篝火重生,可捡回掉落的"灰烬"(类血源血之回响)。

**三种结局**:
1. **净化灰烬** —— 延续初火之王道路,牺牲自己换暂时洁净。
2. **拥抱灰烬** —— 承认灰烬是进化,成为新的灰烬之王。
3. **打破轮回** —— 收集记忆碎片,拒绝二元选择,直面世界核心本体。

## 6. 视觉识别:灰烬与余烬 (Ash & Ember)

**铁律:世界是单色的,直到某物燃烧。**

- **基底层(约 85% 画面)**:冷灰、炭灰、石板蓝灰、去饱和青灰。
- **暖色层(琥珀 / 橙 / 柔金)**:**只用于**记忆碎片、NPC 眼光、篝火光、玩家吸收力量时的灰烬裂痕。
- 暖色 = 记忆 = 价值 = 希望。看到琥珀色,玩家就知道"这里重要"。屏幕全灰 = 危险或失去。
- **形状语言**:有机/侵蚀(自然、安全) vs 锯齿状(腐化、危险)——不靠 UI 就告诉玩家什么是敌对的。
- **区域底色微变**:每区域在灰底上有几乎不可察觉的色调水印(水域暗青、森林橄榄灰)。

**视觉母题**:灰烬飘落(世界在死)、篝火炉火(希望)、锈蚀金属(旧秩序衰败)、枯骨墓碑(遗忘残留)、根系(深处连接)、破碎圣像(信仰崩塌)、琥珀眼光(与初火的联系)。

## 7. 角色阵容(浓缩,横版可直接复用)

### 主角 · 灰化者 (Ashen One)
中性、失忆。窄倒三角剪影:宽兜帽 + 短披风,层叠斗篷收向双靴。炭灰兜帽、氧化铁护臂、一把朴素直剑(暗钢、无装饰金)。两点极小可控的琥珀眼光,只在用技能时亮起、灰烬裂痕泛琥珀。
**反设计**:拒绝英雄板甲、动漫大剑、暴露帅脸、鲜红披风、装饰火焰。

### 核心 NPC(主城情感锚点)
- **铁匠托尔** —— 守着最后可靠炉火的老兵,稳重经济的动作。
- **商人维伊** —— 谨慎的遗物商,记价格胜过记名字。
- **先知伊尔** —— 背负不属于一生记忆的衰弱先知,黑根手杖嵌封印记忆珠。
- **守卫队长玛拉** —— 用太少的人守卫主城,疲惫但纪律。
- **忏悔者奥拉** —— 靠放弃确信而幸存的前信徒。
- **流浪者塞布尔** —— 穿越过禁区的讽刺独行者。
- **矿工幽灵多尔 / 失落灵魂奈姆** —— 区域叙事性幽灵,后者可转敌对。

> NPC 设计哲学:**NPC 不是任务发放机,是玩家知识的镜子**——对话随玩家的发现而变化,没人给你任务标记。

### 敌人 / 精英 / Boss(按区域)
| 区域 | 普通敌人 | 精英 | Boss |
|---|---|---|---|
| 初始墓地 | 墓地灰壳、骨弓遗骸、灰墓猎犬 | 丧钟骑士 | **墓守者** |
| 腐朽森林 | 根行者、孢囊背负者、荆棘爬兽 | 巨根卫士 | **腐朽之母** |
| 沉没教堂 | 溺亡忏悔者、失声唱诗者、锁链执事 | 沉水领唱者 | **堕落主教** |
| 灰烬矿坑 | 矿工遗骸、灰脉甲虫、轨道拾荒者 | 矿坑监工 | **掘灰巨像** |
| 王城废墟 | 王室残兵、断旗幽魂 | 王城处刑官 | **流亡之王** |
| 深渊回廊 | 虚空朝圣者、裂影 | 深渊双生 | **深渊守望者** |
| 火焰神殿 | 焚余侍僧、冻烬守卫 | 火殿香炉骑士 | **火焰守护者** |

## 8. 与现有横版战斗的衔接(关键)

game1 那份被标"作废"的横版战斗文档,其实和你 game3 的头脑风暴**高度同源**,可直接拿来当数值底稿:

| 你的 game3 brainstorm 想法 | game1 横版战斗里的对应实现(可复用) |
|---|---|
| 进攻奖励 / "永远向前打" | 核心幻想就是 **"进攻就是生存,低血是赋能"** |
| 失血返还(Rally) | **Ash Surge**:低血(<25%)+ 攒满灰烬能量触发,1.3x 伤害 + 起手无敌帧 + 战斗中回血(2%/s,上限 30%) |
| 进攻回资源 | **灰烬能量**:每造成 1% 伤害回 1 点,脱战衰减,命中刷新衰减计时 |
| 连击槽 / 鼓励变招 | **连招新鲜度**:同招式连续 ×0.9 衰减,**换招式重置回 1.0x**——逼玩家变招而非搓一招 |
| 帧冻结(hitstop) | 轻击 4 帧 / 重击 8–12 帧,冻结状态机但接受缓冲输入 |
| 防反 / 完美闪避 | game1 选择了**纯闪避**(无格挡):完美闪避 = 闪避最后 3 个无敌帧接触攻击 → 下一击 +15% + 回能。**你 game3 想加防反,是对它的扩展,方向一致** |
| 无敌帧 / 取消系统 | 4 种闪避各有无敌帧表;任意攻击前 1–8 帧可取消进闪避(universal cancel) |
| 武器变形 | game1 未做,是你 game3 的新增点;可挂在"武器升级树"上 |

**具体可抄的帧数据(60 FPS)**:轻击 15–20 总帧 / 重击 29–38(8–12 帧霸体)/ 翻滚无敌帧 4–13 / 快步无敌帧 3–8。完整表见 game1 `02-systems/core-combat-system.md`,需要我整张搬过来就说。

**一个取舍提醒**:game1 当初**砍掉了格挡 / 防反**(理由:闪避取消比防御反应更符合"驾驭灰烬"的流动感)。你 game3 明确想要防反——这没问题,但建议让防反也服务"向前打"(防反回血/回能、强化下一击),否则会和"进攻即生存"的核心打架。
